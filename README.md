# MirWeb — landing

Премиальный тёмный лендинг веб-студии MirWeb: студия делает сайты-витрины для локального бизнеса по модели «бесплатное демо первого экрана → полноценный сайт».

Hero — кинематографичная scroll-сцена: закрытый MacBook-подобный ноутбук (собран кодом, без картинок) открывается при скролле, внутри появляется экран MirWeb, затем zoom-in в экран и переход в full-screen состояние сайта.

## Стек
Vite · React 19 · TypeScript · Tailwind CSS · Framer Motion

## Запуск локально
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # прод-сборка в dist/
```

## Заявки с формы → Telegram
Форма отправляет данные в serverless-функцию `api/lead.ts`, которая шлёт заявку в Telegram. Нужны переменные окружения (на хостинге, не в коде):

| Переменная | Откуда |
|---|---|
| `TELEGRAM_BOT_TOKEN` | бот от [@BotFather](https://t.me/BotFather) |
| `TELEGRAM_CHAT_ID`   | id чата/группы, куда падают заявки |

Без этих переменных (например, на статике) форма мягко деградирует: показывает экран «спасибо» с кнопкой написать в Telegram напрямую.

## Деплой
Рекомендуется **Vercel** (auto-detect Vite + serverless `api/`): импортировать репозиторий на vercel.com, добавить две env-переменные выше — и заявки пойдут в Telegram.
